INPUT输入控件
  • 说明
  • 示例

说明

用于用户数据收集,校验

该控件分为标准模式和左侧带ICON模式两种,关于ICON的应用可以从MEicon中选用

标准模式的input控件必须在其承载的父元素拥有data-type="norm"属性

带ICON模式的input控件必须在其承载的父元素拥有data-type="icon"属性,i 标签为ICON的载体

FILE控件必须在其承载的父元素拥有data-type="file"属性,i 标签为ICON的载体

input的类型分别是：text,password,file

示例

标准模式:

代码示例 
<div data-type="norm">
    <input type="text" placeholder="默认">
</div>
<div data-type="norm" data-state="verify">
    <input type="text" placeholder="验证中......">
</div>
<div data-type="norm" data-state="success">
    <input type="text" placeholder="验证通过">
</div>
<div data-type="norm" data-state="fail" data-label="验证失败原因描述">
    <input type="text" placeholder="验证失败">
</div>
------------------------------------------
请注意：
    该控件的验证状态由 data-state 属性控制；其值分别为：
	verify(验证中，一般用作需发请求的场景下);
	success(校验通过);
	fail(校验失败),其中校验失败的信息由父标签的data-label属性来承载;

左侧带ICON模式：

代码示例 
<div data-type="icon">
    <i class="me-ion-person"></i>
    <input type="text" placeholder="ICON模式">
</div>
<div data-type="icon" data-state="verify">
    <i class="me-ion-o-locked"></i>
    <input type="password" placeholder="ICON模式验证中...">
</div>
<div data-type="icon" data-state="success">
    <i class="me-ion-o-locked"></i>
    <input type="password" placeholder="ICON模式验证成功">
</div>
<div data-type="icon" data-state="fail" data-label="失败原因描述在此!">
    <i class="me-ion-o-locked"></i>
    <input type="text" placeholder="ICON模式验证失败!">
</div>
------------------------------------------
请注意：
    该控件的左侧图标由 i 标签作为载体(加class即可),具体的图标可以在MEicon中挑选
    该控件的验证状态由 data-state 属性控制；其值分别为：
	verify(验证中，一般用作需发请求的场景下);
	success(校验通过);
	fail(校验失败),其中校验失败的信息由父标签的data-label属性来承载;
