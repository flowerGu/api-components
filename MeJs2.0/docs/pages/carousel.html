说明

用于banner，多内容，多产品的滚动展示等场景

该控件只提供基础的HTML结构，实际使用时请根据业务场景对本HTML结构进行CSS的控制

基本的HTML结构如下且约定必须使用本结构,除了UL,LI(必须拥有data-content)外，别的标签的使用不受约束！

轮播的内容必须放到LI里面，原则上轮播内容不受约束

Dom结构:
<div class="xxx">
	<a href="#" class="xxx">prev</a>
	<ul>
		<li data-content>内容</li>
		<li data-content>内容</li>
		<li data-content>内容</li>
	</ul>
	<div class="xxx">翻页在此处，需要的话</div>
	<a href="#" class="xxx">next</a>
</div>
		  
调用方式:
me.require(['carousel'], 'carousel'); //加载arousel.js文件
me.require.ready('carousel',function(){   //关于require的使用请查阅对应的文档
	me.CarouselPlug({
		scrollobj : ".cctv",
		btnNext: ".next",
		btnPrev: ".prev",
		btnGo  : true,
		currentPage : "curs",
		pagesBox : "pages",
		mouseWheel: true,
		visible:1,
		scroll:1,
		circular:true,
		speed: 800, 
		easing : "easeInOutBack"
	})
})

API

属性 说明
scrollobj 滚动视窗对象,class 或者 id
btnPrev 上一个按钮的class名, 比如 btnPrev: ".prev"
btnNext 下一个按钮的class名, 比如 btnNext: ".next"
btnGo 翻页按钮,默认false,传true表示有翻页;
mouseOver 鼠标放上是否停止自动滚动,auto 未false时无效
auto 指定多少秒内容定期自动滚动。默认为空(null),是不滚动,如果设定,单位为毫秒,如1秒为1000;设置自动后btnPrev，btnNext 失效
speed 滑动的速度,可以尝试800 1000 1500,设置成0将删除效果
easing 缓冲效果名称,如:easing: "bounceout",全面支持easing pluin的各种效果
vertical 是否垂直滚动,可选：false,true,默认false
circular 是否循环滚动,默认为true,如果为false,滚动到最后一个将停止滚动
visible 可见数量,可以为小数，如2.5为2.5个li
start 开始的地方,默认是0
scroll 每次滚动的li数量
currentPage 当前"页"className(如果设置了页码的话)
pagesBox 页码的外容器className
beforeStart 滚动开始时回调的函数,可以传入对象参数 beforeStart: function(a) { alert("开始的对象是:" + a)}
afterEnd 滚动结束时回调的函数,使用方法同上
